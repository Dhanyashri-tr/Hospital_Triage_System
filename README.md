# 🏥 AI Hospital Triage System

## 🚀 Overview
This project predicts patient priority based on vital signs such as heart rate, oxygen level, temperature, and pain level.

## ⚙️ Features
- Real-time triage prediction
- Priority scoring system
- AI-based explanation for decisions
- Simple and interactive UI

## 🧠 Decision Categories
- 🚨 TREAT NOW → Critical condition
- ⚠️ MONITOR → Needs observation
- 🕒 WAIT → Stable condition

## 🛠 Tech Stack
- Python
- Gradio

## 🌍 Impact
This system helps hospitals:
- Prioritize critical patients
- Reduce emergency overload
- Improve response time
